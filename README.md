# Errai TodoMVC

A todo manager based on [TodoMVC](http://todomvc.com/) built with Errai and GWT. It uses the following stack:

- Java 8
- GWT 2.8.0
- WildFly 9
- Errai 3.2

## Get Started 

To run this application choose between 

### Development Mode

    mvn gwt:run

### Production Mode

    mvn clean package wildfly:deploy
